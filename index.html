About / Help

This is a fully client-side demo of an Employee Management System. Use the Demo users button at login to create example users quickly.

  • Admins: can add/edit/delete employees, approve/reject leaves, view reports.
  • Employees: can check-in/out, apply for leaves, view own attendance.

For a production app, add a backend (Node/Express + DB), secure authentication (JWT), and server-side validation.
